# Validação de formulário

<img width="100%" src="assets/img/form.png">

## Sobre o Projeto:

Esse é um formulário de preenchimento obrigatório em todos os campos, onde as validações são feitas por javascript.<br> Caso o usuário não preencha ou coloque informação inválida aparecerá um modal notificando e a cada resposta informada a barra de progresso vai concluindo.

## Aprendizado adquerido:

Objetivo em fazer esse projeto era aprimorar o javascript utilizando funções, condições, eventos e aprender sobre o regex.
